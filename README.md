Health & Wellness Tracker
A responsive web application designed to help users take charge of their daily health and lifestyle. Track water intake, sleep, mood, exercises, and plan your day—all from a modern, interactive interface.

Features
User Authentication: Quick login to create a personalized experience.

Physical Details Input: Track and update height and weight securely.

Water Intake Tracker: Log water consumption, get hydration suggestions, and receive alerts for low intake.

Sleep Tracker: Record sleep duration and get health warnings if sleep is insufficient.

Mood Tracker: Track daily mood states and receive tips for mental wellness.

Day Planner: Plan and record daily tasks and routines with an easy-to-use UI.

Exercise Planner: Add exercises, mark them as complete, and receive congratulatory feedback for achieving fitness goals.

Responsive UI + Navigation: Stylish glass-effect nav bar, animated transitions, and dynamic cards for an engaging user experience.

Background Image Rotation: Soft, interactive backgrounds enhance visual interest.

Motivational Feedback: Receive real-time suggestions and positive reinforcement for healthy habits.

Getting Started
Clone the repository.

Open index.html directly in your browser—no build tools needed.

Make sure styles.css and script.js are in the same directory.
